installar npm run all para rodar varios comandos ao mesmo tempo com workspaces
```npm i npm-run-all
```
installar tsup para compilar o projeto typeScript em um unico arquivo
```
npm i -D tsup
```
installar uuid para gerar um id unico
```
npm i uuid
```
para usar o uuid no projeto precisa instalar o @types/uuid
```
npm i -D @types/uuid
```
